# Claude Code Skills Viewer

## 1. 배경

Claude Code는 `SKILL.md` 파일 기반의 Skills 시스템으로 기능을 확장할 수 있습니다. 스킬이 늘어날수록 다음과 같은 불편이 있었습니다:

- 설치된 스킬 목록을 터미널 명령으로만 확인할 수 있었습니다
- 각 스킬의 내용을 파악하려면 파일을 하나씩 열어봐야 했습니다
- 추가/삭제를 위한 전용 도구가 없었습니다

이를 해결하기 위해, **브라우저에서 스킬을 조회·검색·추가·삭제할 수 있는 로컬 웹 뷰어**를 개발했습니다.

---

## 2. 설계 방향

- **Zero-Dependency**: npm 외부 패키지 없이 Node.js 내장 모듈만 사용하여 `npx` 한 줄로 즉시 실행 가능하도록 했습니다
- **단일 파일**: 서버와 클라이언트를 `skills-server.mjs` 하나에 통합하여 배포를 단순화했습니다 (총 1,058줄)
- **Stateless**: DB 없이 파일 시스템을 직접 읽어 항상 최신 상태를 반영합니다

## 3. 주요 기능

### v1.0.0

| 기능          | 설명                                                                                                |
| ------------- | --------------------------------------------------------------------------------------------------- |
| 스킬 조회     | 카드 그리드로 전체 스킬 목록을 표시하며, 클릭 시 마크다운 렌더링으로 상세 내용을 확인할 수 있습니다 |
| 검색          | 이름, 설명, 태그를 대상으로 실시간 필터링합니다                                                     |
| 스킬 추가     | SKILL.md 파일을 드래그 앤 드롭 또는 클릭으로 업로드합니다                                           |
| 스킬 삭제     | 확인 다이얼로그 후 심볼릭 링크와 원본을 함께 제거합니다                                             |
| 테마          | 다크/라이트/시스템 3가지 모드를 지원합니다                                                          |
| 키보드 단축키 | `/`(검색 포커스), `Esc`(닫기)를 지원합니다                                                          |

### v1.1.0 (추가 기능)

| 기능      | 설명                                                                  |
| --------- | --------------------------------------------------------------------- |
| 즐겨찾기  | 스킬에 별표를 달고 필터링할 수 있으며, localStorage에 영속 저장됩니다 |
| 포트 폴백 | 기본 포트(3333) 충돌 시 자동으로 다음 포트를 시도합니다 (최대 5회)    |

### v1.2.0 (크로스 플랫폼)

| 기능                | 설명                                                                                     |
| ------------------- | ---------------------------------------------------------------------------------------- |
| Windows/Linux 지원  | 심볼릭 링크를 절대경로 + junction 방식으로 변경하고, 브라우저 열기를 플랫폼별 분기 처리했습니다 |

### v1.3.0 (프로젝트 스킬 & 테마)

| 기능                | 설명                                                                                     |
| ------------------- | ---------------------------------------------------------------------------------------- |
| 프로젝트 스킬       | 특정 프로젝트의 `.claude/skills/` 스킬을 함께 조회할 수 있습니다                          |
| 프로젝트 관리       | OS 폴더 선택기를 통해 프로젝트를 추가/제거하며, 설정 파일에 영속 저장됩니다               |
| 스코프 배지         | 각 스킬 카드에 Personal/Project 배지를 표시합니다                                        |
| 테마 개선           | Dark는 Dracula, Light는 Minimal Mist 테마를 적용했습니다                                 |
| 빈 설명 처리        | 설명이 없는 스킬에 플레이스홀더를 표시하고 카드 최소 높이를 보장합니다                    |

## 4. 구현 구조

```
skills-server.mjs (801줄)
├── Backend (Node.js)
│   ├── YAML Frontmatter 파서 (정규식 기반 자체 구현)
│   ├── 스킬 로딩 (심볼릭 링크 해석 → SKILL.md 파싱)
│   └── REST API (GET/PUT/DELETE /api/skills)
└── Frontend (인라인 HTML)
    ├── CSS Variables 기반 테마 시스템
    ├── CSS Grid 반응형 레이아웃
    └── Vanilla JS SPA (Lazy 마크다운 렌더링)
```

### API

| 메서드   | 경로                | 설명                  |
| -------- | ------------------- | --------------------- |
| `GET`    | `/api/skills`       | 전체 스킬 목록 (JSON) |
| `PUT`    | `/api/skills/:name` | 스킬 추가             |
| `DELETE` | `/api/skills/:name` | 스킬 삭제             |

---

## 5. 실행 방법

```bash
npx claude-skills-viewer              # 즉시 실행
npx claude-skills-viewer --port 4000  # 포트 지정
```

---

## 6. 결론

Claude Code Skills Viewer는 외부 의존성 없이 단일 파일로 구현된 경량 로컬 도구입니다. `npx` 한 줄로 실행할 수 있으며, 스킬의 조회·검색·즐겨찾기·추가·삭제 기능을 브라우저 UI로 제공합니다. 간결한 설계를 통해 설치 장벽 없이 즉시 사용할 수 있도록 했습니다.
