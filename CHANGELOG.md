# Changelog

## [1.3.0] - 2026-02-24

### Added
- 프로젝트 스킬 지원: 특정 프로젝트의 `.claude/skills/` 스킬을 함께 표시
- 프로젝트 관리 다이얼로그: OS 폴더 선택기로 프로젝트 경로 추가/제거
- 프로젝트 목록 영속 저장 (`~/.claude/skills-viewer.json`)
- 스킬 카드에 Personal/Project 스코프 배지 표시
- 설명이 없는 스킬에 "설명이 없습니다" 플레이스홀더 표시
- 카드 헤더 최소 높이 보장으로 레이아웃 안정화

### Changed
- 테마 변경: Dark → Dracula, Light → Minimal Mist

### API
- `GET /api/projects` — 등록된 프로젝트 목록
- `PUT /api/projects` — 프로젝트 추가
- `DELETE /api/projects` — 프로젝트 제거
- `GET /api/pick-folder` — OS 네이티브 폴더 선택기

## [1.2.0] - 2026-02-24

### Fixed
- Windows/Linux 크로스 플랫폼 지원
  - 심볼릭 링크를 절대경로 + junction 방식으로 변경 (Windows 관리자 권한 불필요)
  - 브라우저 자동 열기를 플랫폼별 분기 처리 (`open`/`start`/`xdg-open`)

## [1.1.0] - 2026-02-24

### Added
- 즐겨찾기(별표) 기능: 스킬 카드에서 ☆ 클릭으로 즐겨찾기 추가/제거
- 즐겨찾기 필터: 헤더 ☆ 버튼으로 즐겨찾기한 스킬만 표시
- 즐겨찾기 상태 localStorage 저장 (새로고침 후에도 유지)
- 검색 + 즐겨찾기 필터 동시 적용 지원
- 포트 충돌 시 자동으로 다음 포트 재시도 (최대 5회)

## [1.0.0] - 2025-02-24

### Added
- 초기 릴리스
- 스킬 목록 조회 (검색, 마크다운 렌더링)
- 스킬 추가 (파일 업로드, 드래그 앤 드롭)
- 스킬 삭제 (확인 다이얼로그)
- 다크/라이트/시스템 테마 지원
- 키보드 단축키 (`/`, `Esc`)
